# Лабораторная работа №1

## Цель: 
1. Изучить основные возможности языка Python для разработки программных систем с интерфейсом командной строки (CLI).
2. Разработать программную систему на языке Python согласно описанию предметной области.
## Задача:
Разработать программную систему на языке Python.<br>
Модель: Сайта по продаже одежды

<em>
Предметная область: онлайн-торговля модной одеждой. <br>
Важные сущности: сайт, покупатель, продавец, товар, категории одежды, корзина покупок, оплата, доставка.<br>
Операции: операция поиска товаров, операция выбора и добавления в корзину, операция оформления заказа и оплаты, операция отслеживания статуса доставки, операция обработки возвратов.
</em>

## Сущности:
Для реализации данной модели были реализованы следующие сущности:

Clothes, ClothingCategory, Customer, Delivery, Seller, ShopingCart, Site.

## Работа программы:
Программа предлагает консольный интерфейс для работы с системой: добавление сайтов, продавцов, товаров, пользователей, а также их просмотр и редактирование. Также для каждой сущности выводится все опреации которые можно произвести с данным объектом.

## Вывод:
В ходе выполнения работы были усвоены основные типы данных языка Python, а также его основные конструкции, такие как циклы, условные операторы, включение модулей, операции ввода, вывода, и создание, и вызов функций. Кроме того изучен принцип создания классов в python. С помощью всего этого была создана программа, реализующая модель сайта по продаже одежды.
